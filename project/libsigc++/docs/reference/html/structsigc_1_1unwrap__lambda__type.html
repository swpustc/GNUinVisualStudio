Detailed Description

template<class T_type>
struct sigc::unwrap_lambda_type< T_type >

Deduces the type of the object stored in an object of the passed lambda type.

If the type passed as template argument is not of lambda type, type is defined to unwrap_reference<T_type>::type.

Deprecated:
Use C++11 lambda expressions instead of libsigc++ lambdas.
